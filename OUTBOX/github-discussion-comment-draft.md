## Variant A: Quota tracking question
> If you want a quick way to see remaining quota across Claude Code Max, Kimi, and Z.ai without uploading keys anywhere, I built a small static dashboard for that:
>

> - Repo: https://github.com/ryan-knowone/quota-dashboard

## Variant B: API key hygiene / privacy
> For anyone wary of uploading API keys to a third-party service: I made a privacy-first quota dashboard that runs entirely in the browser. No server, no localStorage, no keys leaving your machine except to the provider endpoints.
>
> - Demo: https://ryan-knowone.github.io/quota-dashboard/?mock=1
> - Source: https://github.com/ryan-knowone/quota-dashboard
>
> Currently supports Claude Code Max, Kimi, and Z.ai.
